#Work done during degree studies in Virtual Reality Simulators.

Main features I made:

-Health Component: Modular and reusable component that allowed me to apply it to both the player and enemies or traps (like the lava floor), using events to handle death and visual feedback, such as dynamic materials to show vulnerability.

-Localization Dashboard: I worked with this tool, which allowed me to switch between English and Spanish at runtime, allowing the player to see the changes in the user interface without needing to restart the game.

-Difficulty mode: A system based on a single DataTable. Objects consult it to adjust variables values.

-UI & Menus: I implemented a system that allows access to the same Blueprint of options from the Main Menu or the paused game, ensuring consistency.

-Audio system with CUE´s to randomize repetitive actions.

-Moving Traps and lava floor.

-Enemies and items Spawners.

-Options menu with Resolution, Windowmode, framerate, 
color-blindness, language, sound volume and shadows quality

-100% Made in Blueprints.
